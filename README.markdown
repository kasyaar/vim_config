REQUIREMENTS
============
- ruby
- ctags
- cscope

INSTALATION
===========

Using git
---------

  cd ~/

  git clone git://github.com/kasyaar/vim_config.git .vim

  ln -s .vim/vimrc .vimrc

  ln -s .vim/gvimrc .gvimrc
